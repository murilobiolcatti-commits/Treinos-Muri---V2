Como instalar como app

iPhone (Safari)

Compartilhar → Adicionar à Tela de Início

Android (Chrome)

Menu → Adicionar à Tela Inicial ou Instalar Aplicativo

Publicar no GitHub Pages

1. Crie conta em github.com
2. New repository → treinos-muri → Public
3. Faça upload de: index.html, manifest.json, icon-192.png, icon-512.png
4. Settings → Pages → Branch: main
5. Acesse: usuario.github.io/treinos-muri

Netlify Drop (mais rápido)

Acesse app.netlify.com/drop e arraste os 4 arquivos juntos
